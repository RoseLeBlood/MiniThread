Detailed Description

This class encapsulates the idea of a discrete, non-repeating task. Create a work_queue_item when there is something you need to do on a different Thread, but doesn't have to happen periodically. This is a great construct for one off fire and forget tasks.

This is an abstract base class. To use this, you need to subclass it. All of your work_queue_item should be derived from this class. Then implement the virtual on_work function.
